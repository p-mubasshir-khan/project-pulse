# 📖 About Project Pulse

Project Pulse is an engineering initiative focused on building high-quality Android applications with modern architecture, clean code, excellent documentation, and premium user experience.

Instead of creating quick prototypes, the goal is to engineer products that are reliable, maintainable, and enjoyable to use every day.

The first product under Project Pulse is **MyDialer**.

---

# 📱 MyDialer

MyDialer is a premium Android dialer designed to deliver a fast, elegant, and distraction-free calling experience.

The goal is simple:

> Build a dialer that we would personally choose over the default Android dialer.

---
